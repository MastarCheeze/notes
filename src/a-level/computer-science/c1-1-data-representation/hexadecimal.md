# Hexadecimal

> Base 16
> 0-9 and A-F

1 hex digit = 4 binary digits

Advantages
-   Takes up less **screen** space
-   Easier to read, write and understand
-   Easier to debug

## Uses

-   Assembly
-   Machine code
-   Memory address
-   Memory dumps (debugging)
-   ASCII / Unicode
-   HTML Colour codes
-   Error codes
-   MAC address
-   IP address
-   URL
