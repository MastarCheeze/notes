# Hexadecimal

> Base 16
> 0-9 and A-F

1 hex digit = 4 binary digits

Advantages
- :bullet[✅] Takes up less **screen** space
- :bullet[✅] Easier to read, write and understand
- :bullet[✅] Easier to debug

## Uses

- Assembly
- Machine code
- Memory address
- Memory dumps (debugging)
- ASCII / Unicode
- HTML Colour codes
- Error codes
- MAC address
- IP address
- URL
